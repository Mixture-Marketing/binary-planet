# APPENDIX G — Pierwsze 5 katalogów PL do auto-citation (realistyczna lista)

Research obalił mit o 30 aktywnych katalogach. Realnie:

**Must-have (każdy klient):**
1. **Google Business Profile** — fundament, OAuth API, auto-posting + reviews fetch
2. **Bing Places** — API dostępne (`Bing Maps Places API`), bulk import via CSV
3. **Apple Business Connect** — API w 2026, free, ważne dla iOS users
4. **Panorama Firm** (panoramafirm.pl) — manual submission, ale stabilny ruch
5. **pkt.pl** — manual submission, największa polska baza firm

**Branżowe (per industry):**
- **Medical:** ZnanyLekarz (manual submission, weryfikacja PWZ)
- **Beauty:** Booksy (jest CMS-like sam w sobie, NAP automatyczny po założeniu profilu)
- **Legal/Professional:** Oferia.pl
- **Architects:** Archiweb (jeśli aktywny), Houzz PL (słabsza penetracja)

**Nice-to-have (Local SEO Pro):**
- Aleo, Yelp PL (niski ruch, ale sygnał konsystencji NAP)

**UNIKAĆ:**
- GoldenLine (zamykany)
- mechanik.pl, autoreklama.pl (nie są NAP directories)
- Foursquare/Swarm (umierające w PL)

**Strategia:**
- Auto-submit (API): GBP, Bing Places, Apple
- Half-auto (template + manual click): Panorama Firm, pkt.pl, branżowy (~5–10 min/klient batch, jednorazowo przy onboardingu)
- Tracking statusu w `citations` table
